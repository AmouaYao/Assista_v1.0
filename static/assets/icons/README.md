# Icônes PWA pour E-Menus

Ces icônes sont utilisées pour l'application Progressive Web App (PWA).

## Tailles requises
- 72x72px - icône de base
- 96x96px - icône moyenne
- 128x128px - icône grande
- 144x144px - icône pour iPad
- 152x152px - icône pour iPhone/iPad
- 192x192px - icône pour Android
- 384x384px - icône HD
- 512x512px - icône maximum

## Instructions
Pour générer les icônes, utilisez une image de base (logo E-Menus) et redimensionnez-la aux différentes tailles.

Format recommandé : PNG avec fond transparent.
